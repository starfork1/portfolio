# Моє портфоліо

Це персональний веб-сайт-портфоліо, створений з використанням HTML, CSS та JavaScript.

## Структура проекту

- index.html — головна сторінка
- style.css — стилі сайту
- script.js — JS для інтерактивності
- images/ — каталог з картинками
- .gitignore — ігноровані файли
- README.md — опис проекту
- LICENSE.txt — ліцензія
- CONTRIBUTING.md — інструкції для контриб’юторів
- .env.example — приклад змінних середовища
